# ChessKing
Virtual local chess game with smart features

## Milestone 1 - Single Player Implementation

- [x] User can move his pieces in any legal way possible
- [x] User can click and hold to drag chess pieces
- [x] Available legal moves can be shown to the user
- [x] User can castle
- [ ] User can promote a piece
- [x] User's movements are mirrored on the opponent's board
